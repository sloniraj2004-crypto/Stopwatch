# ⏱️ Stopwatch Web App

A simple and responsive Stopwatch Web Application built using HTML, CSS, and JavaScript.  
It allows users to start, pause, resume, and reset the timer with a clean UI.

---

## 🚀 Features
- ▶️ Start stopwatch
- ⏸️ Pause functionality
- 🔁 Resume timer
- 🔴 Reset stopwatch
- ⏱️ Real-time tracking
- 📱 Responsive design

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript

---

## 📂 Project Structure
Stopwatch/
│── index.html

│── script.js  
│── images/  

---

## ▶️ How to Run
1. Download the project  
2. Open `index.html` in browser  
3. Start using stopwatch
